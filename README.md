# m1_benchmark

Some benchmarks of MacOS M1 (Apple Silicon)

## Installation

with [basher](https://github.com/basherpm/basher)

	$ basher install pforret/m1_benchmark

or with `git`

	$ git clone https://github.com/pforret/m1_benchmark.git
	$ cd m1_benchmark

## Usage

	> m1_benchmark.sh [action]
